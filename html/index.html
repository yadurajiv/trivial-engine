From Complex to Trivial

The key focus of the complete framework is to make complex subsystems easier to manage and build for: we have an accessible camera system, an image manager, font manager, audio manager, event manager, and other resource managersin development

The event manager lets you subscribe any object to a huge range of system or custom events. The camera systems exposes trivial functions to simplify look-ats and camera movementsimplemented

The engine smartly uses spritesheets to work with animations. The engine can create multilayered maps with tilemapsimplemented

An advanced 2d rendering system that exposes sprites to image blending and a multi-layered scene with parallax effectimplemented

Nifty tools are built right in

The engine has a range of nifty tools built right in. These take care of those subtle, routine things that developers dont think much ofin development

There is a tool that captures screenshots from the game, another that captures video and audio from the game and saves it to the diskplanned for release

The engine also includes a complete GUI class that includes labels, label manipulation, buttons, range of input types. The system also supports custom fontsin development

Made for Speed

We dont call it Trivial for no reason: getting started and getting an app up and running is as simple as a writing two function callsimplemented

The engine works with hardware accelerated audio and video allowing you to squeeze out that performance for your game. Different resource managers efficiently handle memory and resources, allowing you to scale up to thousands of sprites in a single scenein development

System configuration is a simple XML based system with rapid turnaround processesimplemented

The engine exposes simple translation and manipulation functions for working with sprites and basic physicsimplemented

The audio manager extends streaming support and looping tracks and allows to manipulate and transform sound with exposed information like sample rates, channel counts, playing position and actual sample data through single function callsimplemented

Many Platforms, Many Choices

Easily develop and deploy for Windows, Mac and Linux. Trivial Engine is proud to be cross-platform and it will reach out to newer platforms in futureimplemented

The engine supports a range of input devicesimplemented

The engine exposes tonnes of Helper classes and functions for all of its built-in objects in a thorough object-oriented universeimplemented

We support a huge range of image, audio, video, fonts and resource formats. Data interchange goes through XMLs and JSON is coming soon. If there is a format worth using, we support it!in development

Social from the Go

Trivial Engine is part of the Trivial Stack and is completely integrated with Trivial Analytics and Sense right out of the boxplanned for release

There is an extensive Network manager that will make a client/server system easy to deploy and also extend an easy to use IRC client systemplanned for release

Easy to find your way Around

With Trivial Engine, you dont have to stumble around in developer hell. The engine is heavily documented in an extensive documentation built with Doxygenin development

It also comes with detailed examples and demos explainging every feature and tool in the engineplanned for release
